🌐 FAIR-TALE B2B 온라인 박람회 플랫폼 사업계획서 (초안)
1. 사업 개요
사업명(가칭): Fair-Tale K-D Biz Match

플랫폼 형태: B2B 온라인 전시·박람회·제품 기술 매칭 플랫폼

사업 목적: 물리적 전시회 참여가 제한적인 중소기업들이 디지털 기반으로 제품 홍보, 기술 소개, 파트너 매칭을 효율적으로 수행할 수 있도록 지원

초기 타깃 시장: 한국 ↔ 독일

향후 확장 대상: 유럽 전역 → 아시아 → 미국

✅ 보너스: 비용 절감 팁
번역은 DeepL Pro API (월 수십 유로)로 충분히 베타 가능

WebRTC는 자체 구현보다 Daily.co, Agora, Twilio Video 등 사용 추천

통합 인증은 Firebase Auth + 기업정보 수동검증 조합으로 빠르게 처리 가능
