## ✅ 0일차: 소켓 연결과 대화방 관리 🔌 **[완료]**
### 1. 소켓 연결 순서 ✅
1) 첫 번째: 소켓 준비하기 ✅
   - 소켓을 만들기 전에 설정을 준비해요
   - 서버 주소, 연결 시간, 재연결 횟수를 정해요
   - 연결이 끊어졌을 때 어떻게 할지 계획을 세워요

2) 두 번째: 소켓 연결하기 ✅
   - 준비한 설정으로 소켓을 만들어요
   - 연결이 성공했는지 확인해요
   - 연결이 실패하면 다시 시도해요 (최대 5번까지)

3) 세 번째: 연결 상태 확인하기 ✅
   - 연결됐다고 알려주기
   - 끊어졌다고 알려주기
   - 에러가 났다고 알려주기

### 2. 대화방 입장 순서 ✅
1) 첫 번째: 방 입장 전 확인 ✅
   - 소켓이 잘 연결되어 있는지 확인해요
   - PIN 번호가 맞는지 확인해요
   - 카메라와 마이크 사용 허락을 받아요

2) 두 번째: 방 입장하기 ✅
   - 소켓으로 방 입장 신청을 해요
   - 방에 잘 들어갔는지 확인해요
   - 다른 사람들에게 내가 들어왔다고 알려요

3) 세 번째: 방 관리하기 ✅
   - 연결이 끊어지면 자동으로 다시 들어가요
   - 나갈 때는 깔끔하게 정리해요
   - 에러가 나면 친절하게 알려줘요

### 3. 연결 관리 규칙 ✅
1) 기다리는 시간 ✅
   - 소켓 연결: 5초
   - 방 입장: 3초
   - 재연결 시도: 2초

2) 순서 지키기 ✅
   - 소켓 연결 → 방 입장 → 미디어 시작
   - 하나가 실패하면 처음부터 다시
   - 모든 정리는 역순으로

3) 에러 처리 ✅
   - 문제가 생기면 바로 알려주기
   - 자동으로 해결할 수 있으면 시도하기
   - 안되면 사용자에게 친절하게 설명하기

### 4. 정리 순서 ✅
1) 방에서 나갈 때 ✅
   - 미디어 중지
   - 방 나가기 신호 보내기
   - 소켓 연결 정리하기

2) 에러가 났을 때 ✅
   - 진행중인 것 모두 중지
   - 깨끗하게 정리하기
   - 다시 시작할 준비하기
